# ToDos_unPlatforms
Created with CodeSandbox
# Tools Used
The main framework used in building the To-Do List website is ReactJs(Routing, components,UseEffect, UseState Hooks, and React icons) & Bootstrap and its components like Carousel, Card, Buttons, Navbar etc.

# Create Task
On Clicking the Create Task button, a Modal will appear as shown below, In which the TaskName, Description & DeadLine is set and click on Create button.
<img width="960" alt="image" src="https://user-images.githubusercontent.com/91959504/189542188-eb31854f-83a5-4ae0-9dab-e4005167664e.png"> 

# Filling the Modal
